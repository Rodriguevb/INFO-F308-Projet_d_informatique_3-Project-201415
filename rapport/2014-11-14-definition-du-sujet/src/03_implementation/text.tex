Pour résoudre ce problème, nous allons utilisé un algorithme linéaire comme celui présenté dans l'article du European Journal Operational Research \cite{mainref}. 
L'espace sera organisé sous forme de grille afin de facilité les calculs.


Nous tenterons de développer un programme C++ facile d'utilisation qui permettra de fournir un placement optimisé des sources de lumière d'une route en fonction des données reçues.
L'algorithme se basera sur une grille construite à partir d'un réseau routier quelconque et placera de façon optimale les sources de lumière.

Valeurs prises en compte par l'algorithme :
- Type de route ;
- Fréquentation de la route ;
- Vitesse maximale autorisée ;
- Coéfficient de dangerosité ;
- Météo ;
- Lumière du soleil (heure) ;
- Budget à disposition.

Résultat de l'algorithme :
- Matériel utilisé (économique et écologique) ;
- Intensité de la lampe ;
- Distance entre les sources des lumières ;
- Hauteur des sources des lumières ;
- Coût d'installation ;
- Coût d'utilisation (graphe en fonction de l'heure) ;
- Coût de maintenance.
